Vamos apreder a redimencionar as imagens nessa aula.

Nesta aula vamos aprender a redimencionar imagens utilizando as tags "picture" e "Source"
1º Passo > Vamos adicionar a maior imagem e logo após temos que delimitar o tamanho dela para 1050px. (Significa que o maximo para essa imagem continuar em evidencia é 1050px), Se o valor for menor do que 1050px a imagem é alterada para imagem Média.
2º Passo > Repetindo o passo anterior e delimitando a imagem média em 750px, assim que o usuário diminuir a pagina ela aciona rapidamente a imagem Pequena
